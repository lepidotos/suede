Terminal properties

The terminal has a large number of properties which can be set via object methods.

  • - set absolute terminal size.

  • - enable/disable blinking cursor.

  • - enable/disable bell.

  • - read bell state.

  • - enable/disable auto-scroll on keystroke

  • - enable/disable auto-scroll on output

  • - enable/disable DEL-BS key swap.

  • - set characters which match words in selections

  • - set the color palette

  • - restore default color palette.

  • - set scroll-back buffer size.

  • - set fonts.

  • - set bold/normal font by name.

  • - set background picture/transparency options.

  • - query compiled capabilities.

These commands change the behavior at runtime.

  • - Make the mouse pointer visible.

  • - Make the mouse pointer invisible.
