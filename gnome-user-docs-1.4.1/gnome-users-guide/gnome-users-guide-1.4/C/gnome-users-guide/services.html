Eazel Services

Eazel Services are Internet-based tools designed to simplify system management for Linux users. Nautilus and Eazel Services work together to make your life easier.

Eazel Services include

  • Eazel Online Storage, for file backup

  • File sharing capability

  • Eazel Software Catalog, for easy installation of software titles

  • The Nautilus installer

Registering with Eazel

When you first launched Nautilus, you were asked if you wanted to register for Eazel Services. If you registered at that time, click the Services button in the Nautilus window and log in, using your user name and password.

If you have not already registered, click the Services button to see a registration screen. Follow the steps for registering.
