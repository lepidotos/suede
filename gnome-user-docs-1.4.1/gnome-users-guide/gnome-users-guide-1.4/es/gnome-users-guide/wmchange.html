Cambiando de gestor de ventanas

Cada vez que quiera cambiar de gestor de ventanas, deberá usar el aplique C del gestor de ventanas en el Centro de Control. Puede leer más acerca de este aplique C en la sección de nombre Aplique C Gestor de ventanas en el capítulo de nombre El Centro de Control de GNOME

IMPORTANTE
 

Tenga siempre presente que el gestor de ventanas que decida usar puede no ser compatible con GNOME, e impedirle beneficiarse de algunas de las funcionalidades de GNOME en caso de usarlo.
