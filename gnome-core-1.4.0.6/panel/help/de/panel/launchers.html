Starter

Starter sind im wesentlichen Knöpfe, die sich in Ihrem Panel befinen und die eine Applikation starten oder einen Befehl ausführen, wenn man auf sie klickt. Ein solcher Starter kann jedes beliebige Icon verwenden und hat eine eigene Minihilfe, die angezeigt wird, wenn mann den Cursor über den Starter bewegt.

Um die Eigenschaften eines Starters zu ändern, z.B. sein Icon oder seinen Namen, rechtsklicken Sie auf ihn und wählen Eigenschaften im Kontext-Menü. Dies bringt den Starter-Eigenschaften-Dialog zum Vorschein, der in Figure 7 gezeigt wird. Dieser Dialog wird ebenfalls benutzt, wenn Sie einen neuen Starter hinzufügen (siehe the Section called Objekte zum Panel hinzufügen). Darüberhinaus macht GNOME intern keinen Unterschied zwischen Startern und Menüeinträge: beide sind einfach verschiedene Darstellungsformen derselben Sache. Sie können jeden Menüeintrag auf dem Panel plazieren und er erscheint als Starter. Daher sind alle unten angegebenen Informationen für Starter und Menüeinträge gleich.

Figure 7. Dialog "Starter erzeugen/Starter-Eigenschaften"

Dieser Dialog hat zwei Reiter: Einfach und Erweitert. Im Einfach-Reiter können Sie folgendes angeben:

  • Name — dies ist der Name der Applikation, zum Beispiel GNOME-Terminal. Dieser Name wird benutzt, wenn Sie den Starter später in ein Menü einfügen.

  • Kommentar — dies ist ein kurze Beschreibung, wozu die Applikation gut ist, zum Beispiel GNOME-Terminalemulator. Dies wird für die Minihilfe benutzt.

  • Befehl — der eigentliche Befehl, der die Applikation startet, zum Beispiel gnome-terminal.

  • Typ — sollte Applikation sein; ändern Sie ihn nicht, wenn Sie nicht etwas anderes als einen Applikations-Starter erzeugen wollen.

Der Einfach-Reiter zeigt außerdem das für den Starter verwendete Icon an; wenn kein Icon angegeben wurde, wird ein Standard-Icon verwendet. Um das Icon zu ändern, klicken Sie es einfach an um den Icon-Browser zu starten.

Zum Schluß kann die Applikation auch noch ausschließlich innerhalb eines Terminals ablaufen — das heißt, sie erzeugt keine eigenen Fenster — in diesem Fall sollten Sie den In Terminal starten Knopf anwählen. Falls unsicher, lassen Sie ihn ausgeschaltet.

Der Erweitert-Reiter wird unten gezeigt. Er ist für fortgeschrittene Benutzer gedacht; meistens brauchen Sie keine dieser Einstellungen zu verändern.

Figure 8. Dialog "Erweiterte Starter-Eigenschaften"

In diesem Reiter können Sie folgendes setzen:

  • Dies zuerst testen — hier können Sie einen Befehl eingeben, und GNOME überprüft zunächst, ob dieser Befehl ausgeführt werden kann. Andernfalls wird der Starter (oder der Menüeintrag) nicht angezeigt, selbst wenn Sie ihn zu einem Panel oder Menü hinzugefügt haben. Diese Option ist hauptsächlich für Leute gedacht, die GNOME-Distributionen vorbereiten: das standardmäßige GNOME-Hauptmenü enthält z.B. einen Eintrag für das TkRat Mailprogramm, aber er wird nicht angezeigt wenn Sie TkRat nicht installiert haben.

  • Dokumentation — zur Zeit unbenutzt.

  • Name/Kommentarübersetzung — hier können Sie Übersetzungen von Name und Kommentar in andere Sprachen eingeben. Für das GNOME-Terminal, sind z.B. die spanischen Übersetzungen für den Namen Terminal UNIX de GNOME sowie Emulador de terminal GNOME für den Kommentar. Wenn ein Benutzer nun beim Anmelden als seine Sprache Spanisch einstellt (dies geschieht durch Wahl der jeweiligen Sprache im Sprachen-Menü in GNOME-Loginmanager (GDB)), erscheint Terminal UNIX de GNOME im Menü und Emulador de terminal GNOME als Minihilfe. Selbstverständlich bleibt der eigentliche Befehl, der das Terminal startet unverändert.

    Um eine neue Übersetzung hinzuzufügen, geben sie den 2-buchstabigen Sprachcode und die Übersetzungen von Name und Kommentar in die leeren Felder unterhalb der Liste der gegenwärtigen Übersetzungen ein und klicken Sie den Hinzufügen/Ändern-Knopf. Um eine bestehende Übersetzung zu ändern, wählen Sie sie in der Liste aus, ändern die gewünschten Felder und klicken den Hinzufügen/Ändern-Knopf. Um eine bestehende Übersetzung zu entfernen, wählen Sie sie aus und klicken Sie den Entfernen-Knopf.

    Sie finden eine Liste aller von GNOME unterstützter Sprachen zusammen mit ihrem 2-buchstabigen Sprachcode in GNOME Frequently Asked Questions.

Alle im Starter-Eigenschaften-Dialog werden wirksam, wänn Sie den Übernehmen oder OK-Knopf klicken. Mit OK schließen Sie diesen Dialog, Übernehmen übernimmt die Änderungen und erlaubt Ihnen das weitere Editieren.
