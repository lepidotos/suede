作者

GNOME パネルは、多くの GNOME 開発者に より作成されました。パネルについてダイア ログで一部のリストを見ることができます。ところで、パ ネルについてダイアログにでてくる動物の名前を知りたい? あれは、"Gegl" と呼ばれてて、そいつのWeb ページがあります。

批評、提案、バグ報告は全て、GNOME バグトラッキングデータベースに送ってく ださい。バグ報告を送る手順については、オンラインの http://bugs.gnome.org/Reporting.html (bug-buddy)で知ることができます。バグレポート の送信には、メインメニュー中の ユーティリティーサブメニューにある、 バグレポートツールを使うこともできま す。

本マニュアルは、Dave Mason(<dcm@redhat.com>)、Dan Mueth(<d-mueth@uchicago.edu>)、Alexander Kirillov (<kirillov@math.sunysb.edu>)が作成しました。このマ ニュアルに関する批評や提案は、GNOME Documentation Project の <docs@gnome.org> に送るか、GNOME Documentation Status Tableを使って下さい。(訳注: 日本語訳に関して は <gnome-translation@gnome.gr.jp>へ。)
