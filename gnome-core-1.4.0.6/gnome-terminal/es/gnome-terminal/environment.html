Entorno

El terminal de GNOME crea las siguientes variables:

  • TERM

    Esta variable tiene el valor «xterm» por omisión. Algunas implementaciones de curses y algunas aplicaciones basadas en curses puede que necesiten que el valor sea xterm-color, color-xterm o xterm-debian para que puedan ejecutarse en color. Tal vez tenga que cambiar el valor de esta variable a uno de estos valores si su aplicación no aparece en color. Podrá hacer esto utilizando la opción de la línea de órdenes --termname.

  • COLORTERM

    Esta variable tiene el valor «gnome-terminal». Varias aplicaciones (especialmente las que utilizan Slang) utilizan esta variable para determinar si el terminal admite colores.

  • WINDOWID

    Esta variable contiene el identificador de la ventana X11. La utiliza a veces la aplicación para modificar los contenidos de la ventana o para hacer algún efecto especial.
