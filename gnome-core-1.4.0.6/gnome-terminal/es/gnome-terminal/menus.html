Menús

GNOME Terminal proporciona una serie de características de control accesibles a través de un menú situado en la parte superior de la ventana del terminal y a un menú desplegable que aparece al pulsar el botón derecho del ratón y la tecla CTRL.

Menú archivo

Nuevo terminal

Crea un nuevo terminal. Es una forma eficiente de crear varios terminales, ya que se usan menores recursos de sistema que en la creación de una copia separada de GNOME Terminal. (Para usuarios experimentados: la nueva ventana de terminal creada de esta forma pertenece al mismo proceso, con el mismo PID que el programa original. Sin embargo cada ventana ejecutará su propia línea de órdenes.)

Esconder la barra de menú

Esconde la barra de menú del terminal, creando un terminal más pequeño y compacto. Puede hacer reaparecer la barra de menú utilizando el menú desplegable.

Cerrar el terminal

Cierra el terminal actual. También cierra cualquier otro terminal que haya sido abierto mediante la línea de órdenes de este terminal. Si éste es el el último terminal en cerrarse, entonces el programa GNOME Terminal finaliza.

Menú editar

Pegar

Pega la selección actual o el contenido del porta papeles en el terminal. Esto también puede conseguirse utilizado el botón central del ratón.

Menú configuración

Preferencias ...

Muestra el formulario preferencias, descritas en la sección opciones configurables.

Reiniciar el terminal

Reinicia los parámetros del terminal. Esta operación no limpiará la pantalla ni desplazara el cursor, aunque cualquier presentación posterior en el terminal se hará con la letra y los atributos por omisión.

Reiniciar y limpiar

Reinicia los parámetros del terminal, y además borra la pantalla. Todos los tipos de letra y los atributos son reiniciados.

Selector de colores...

Puede utilizarse para arrastrar y soltar colores en el terminal. Vea la descripción en la sección de nombre Configuración del color.

Menú ayuda

Para llegar a este documento desde el terminal de GNOME Terminal.

Acerca de...

Un pequeño reconocimiento a todos chicos que trabajaron duro para hacer que el terminal GNOME sea una realidad.

Menú desplegable

Podemos hacer aparecer el menú desplegable pulsando el botón derecho del ratón y la tecla CTRL simultáneamente. (En algunas ocasiones no es necesario pulsar la tecla CTRL para obtener el menú desplegable, pero no debe suponer que siempre sea así.). Este menú es simple atajo para algunas de las funciones del menú principal; esto es especialmente útil cuando el menú principal está oculto. Están disponibles las siguientes opciones:

Nuevo terminal ...

Crea un nuevo terminal.

Preferencias ...

Muestra el formulario preferencias, descrito en la sección de nombre Opciones configurables.

Mostrar la barra de menús

Esconde o muestra la barra de menú principal en la parte superior de la ventana de terminal.

Teclado seguro

Esta opción fuerza el envío directo de todas las entradas de teclado al terminal, puenteando cualquier otra cosa. Esto es útil en aquellas (raras) situaciones en las que varios usuarios usan el mismo servidor X, y el servidor X no esta usando métodos apropiados de autenticación, haciendo posible que otros usuarios capturen lo que este tecleando. En estas circunstancias, es aconsejable conmutar al modo "teclado seguro" antes de teclear cualquier información sensible, como las claves.

Reiniciar el terminal

Reinicia los atributos del terminal (tipografía y fondo) sin borrar la pantalla.

Reiniciar y Limpiar

Reinicia los atributos del terminal y borra la pantalla.

Abrir en un navegador

Si el ratón apunta a un URL (Por ejemplo, una dirección Internet) cuando se invoca el menú desplegable, se dispondrá de esta opción para arrancar un navegador y ver el URL. Usar la tecla CTRL y el botón derecho es un atajo a este ítem de menú.
