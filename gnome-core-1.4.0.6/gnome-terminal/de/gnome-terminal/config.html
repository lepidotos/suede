Konfigurierbare Optionen

Konfigurierbare Optionen können Sie im Eigenschaften-Dialog ändern, den Sie über Einstellungen ... im Einstellungen-Menü erreichen.

Allgemeine Optionen

Während die meisten der Optionen selbsterklärend sind, werden einige davon nachfolgend etwas detailierter beschrieben.

Terminalklasse

Setzt die Terminalklasse für die aktuelle Konfiguration oder das aktuelle Terminal. Einzelheiten dazu finden Sie unter Klassen.

Zeichenmenge für Wortmarkierung

Definiert eine Menge von Zeichen (oder Zeichenklassen), die als 'Wortzeichen' betrachtet werden sollen. Sie wird benutzt, wenn Sie Text wortweise markieren.

Hintergrundtyp

Im Grafik-Reiter können Sie den Hintergrund für das Terminal bestimmen. Mögliche Hintergrundtypen sind:

  • Keiner — in diesem Fall benutzt das GNOME-Terminal die Hintergrundfarbe aus dem Farben-Reiter.

  • Hintergrundpixmap — diese Option ermöglicht Ihnen die Wahl eines Hintergrundbildes. Sie können den Suchen ...-Knopf dazu verwenden, das Bild auszuwählen.

  • Transparent — diese Option macht das GNOME-Terminal "transparent". Es ist nicht wirklich transparent, es simuliert nur den Hintergrund Ihres Desktops wie ein Chamäleon. Erwarten Sie jedoch nicht, andere Fenster durch das GNOME-Terminal hindurch zu sehen.

Falls Sie Pixmap oder Transparent als Hintergrund wählen, können Sie auch die Option Hintergrund sollte schattiert sein wählen, welches das Hintergrundbild "abschirmt" oder "verdunkelt".

Farbkonfiguration

Das GNOME-Terminal unterstützt alle Farberweiterungen, die das xterm-Programm verwendet. Der Farben-Reiter ermöglicht Ihnen, diese Farben anzupassen.

Farbschema

Einige Applikationen sind in der Lage, die Farbfähigkeit des Terminals zu verwenden. Die Interaktion zwischen der Applikation und dem Terminal erfolgt auf eine Art und Weise, in der die Applikation eine von sechzehn bekannten Farben (ANSI-Farben), wie etwa "blau" oder "leuchtend rot", anfordern kann: das Mail-Programm mutt kann zum Beispiel wichtige Nachrichten in rot darstellen. Es ist jedoch Aufgabe des Terminals, zu bestimmen, welche Farbe denn nun genau für "leuchtend rot" oder "blau" verwendet wird. Die Zusammenstellung dieser sechzehn Farben nennt man eine Palette oder ein Farbschema.

Das GNOME-Terminal unterstützt vier Farbschemata: das Linux-Konsole-Schema, das Farbiges XTerm-Schema, das rxvt-Schema und ein frei zusammenstellbares Schema. Diese Schemata sind alle etwas verschieden voneinander: zum Beispiel benutzt die Linux-Konsole hellgrau statt "weiß", während rxvt wirklich weiß benutzt.

Wenn Sie das Eigene-Schema gewählt haben, können Sie alle 16 Farben (genaugenommen 8 Farben und 8 leuchtende Farben) anpassen, indem Sie auf die Farbwähler-Knöpfe klicken.

Vorder-/Hintergrundfarbe

Hier können Sie die Standard Vorder- und Hintergrundfarbe festlegen, die vom Terminal benutzt wird, wenn keine Farben von der im Terminal ablaufenden Applikation angefordert wurden.

Folgende Optionen stehen zur Verfügung: Weiß auf schwarz, Schwarz auf Weiß, Grün auf Schward, Schwarz auf Hellgelb und Eigene Farben. Wie diese wirklich erscheinen, hängt jedoch von der gewählten Palette ab (für die Optionen Weiß auf schwarz und Schwarz auf weiß). Eigene Farben ermöglicht Ihnen, die standardmäßig benutzte Vorder- und Hintergrundfarbe individuell und unabhängig von der Palette festzulegen.

Zusätzlich können Sie eine Farbe aus der rechteckigen Box nahe des Farbrades aus jedem GNOME-Farbwähler wählen und sie auf Terminal fallenlassen. Wenn Sie die Farbe auf einer leeren Fläche fallenlassen, wird sie als Hintergrundfarbe benutzt, andernfalls als Vordergrundfarbe. Dies ändert außerdem automatisch den Vorder-/Hintergrundtyp auf Eigene Farben und setzt die gewählte Vorder- bzw. Hintergrundfarbe.

Rollen

In diesem Reiter können Sie die Position der Rollbalken und die Anzahl der Zeilen festlegen, die zum Rollen im Speicher behalten werden sollen.
