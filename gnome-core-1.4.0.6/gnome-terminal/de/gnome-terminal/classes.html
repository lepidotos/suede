Terminalklassen

Das GNOME-Terminal unterstützt verschiedene Konfigurationprofile, so daß es sich für verschiedene Zwecke einsetzen läßt (verschiedene Hintergrundfarben, Vorhandensein und Position von Rollbalken usw.). Um diese einzurichten, starten Sie den Eigenschaften-Dialog aus dem Einstellungen-Menü. Machen Sie dann alle gewünschten Änderungen (sie wirken sich auf Ihr gegenwärtiges Terminal aus, so daß Sie sehen können, wie es aussehen wird) und geben einen neuen Namen in das Feld Terminalklasse ein, bevor Sie das Dialogfeld schließen.

Um beim Programmstart eine bestimmte Terminalklasse zu aktivieren, können Sie den Kommandozeilenparameter --tclass benutzen.
