Description

GNOME Terminal is a terminal emulator program that is part of the GNOME project. It provides access to the Unix shell on the GNOME environment.

GNOME Terminal supports colour display (see the Section called Colour configuration for details) and provides mouse support to applications that are aware of mouse events for xterm-like terminals.
