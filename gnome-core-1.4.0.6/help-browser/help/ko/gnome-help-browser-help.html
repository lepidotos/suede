그놈 도움말 브라우저 문서

그놈 도움말 브라우저는 당신 컴퓨터에 있는 여러종류의 문서들에 대한 인터페이스를 제공할 뿐만 아니라 인터넷을 통해 다른 컴퓨터로 부터도 문서를 가져올 수 있습니다. 이 짧은 문서는 그놈 도움말 브라우저의 주요 기능을 설명하고 있습니다.

명령행 옵션

      -d, --debug                디버그 레벨, 디버그 레벨을 올리면 더 많이 출력
  -h HEIGHT                  창 높이
  -w WIDTH                   창 넓이
  -x X                       창의 X 좌표
  -y Y                       창의 Y 좌표
      --display=DISPLAY      사용할 X display
      --no-xshm              X의 공유 메모리 확장 기능 사용않함
  -?, --help                 이 도움말 목록 출력
      --usage                짧은 사용 설명 출력
  -V, --version              프로그램 버젼 출력

메뉴

그놈 도움말 브라우저는 응용프로그램 창의 제일 위에 여러 개의 메뉴를 가지고 있습니다. 각 메뉴의 동작은 보기만 해도 짐작하실 수 있을 것입니다.

툴바

툴바는 도움말 브라우저를 좀더 편하게 사용할 수 있도록 여러 개의 버튼을 가지고 있습니다. 다음과 이전 버튼은 당신이 방금 전에 방문했던 페이지들을 볼 수 있도록 해 줍니다. 도움말 버튼은 이 도움말을 보여줍니다. 히스토리 버튼은 당신이 방문했던 사이트 정보를 보여주며, 책갈피 버튼은 이전에 당신이 표시해 두었던 책갈피 목록을 보여줍니다.

URL 입력 박스

당신은 입력 박스를 통해 URL을 직접 입력하실 수도 있습니다. 도움말 브라우저는 다음과 같은 표준 방식과 매직 URL 형식을 지원합니다:

http:
HTTP 송신 프로토콜을 이용하여 문서 접근

예제:
  • http://www.gnome.org
  • http://www.gnome.org/index.shtml
  • http://www.gnome.org/index.shtml#majic
file:
로컬 파일시스템의 파일 접근

예제:
  • file:/home/msf/maji12.html
  • file:/usr/man/man1/ls.1
  • file:/usr/src/gogo/README
whatis:
주어진 문자열로 문서 검색

가능한 형식:
  • whatis:ls
  • whatis:xman
toc:
가능한 문서들의 목차를 적재

가능한 형식:
  • toc:
  • toc:man
  • toc:info
  • toc:ghelp
man:
매뉴얼 페이지 적재

예제:
  • man:ls
  • man:ls(1)
info:
그누 인포 페이지 적재

예제:
  • info:emacs
  • info:emacs#Top
ghelp:
응용프로그램의 그놈 도움말 적재
노우트:현재 이 형식의 문서는 그리 많지는 않습니다.

예제:
  • ghelp:help-browser

책갈피 창

책갈피 창은 현재 가능한 모든 책갈피를 보여줍니다. 책갈피를 추가하려면 파일 메뉴의 책갈피 추가 옵션을 사용하십시오.

히스토리 창

히스토리 창은 당신이 그동안 방문했던 사이트 목록을 보여줍니다. 설정 메뉴에서 최대 길이를 조정하실 수 있습니다.
