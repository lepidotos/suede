A GNOME súgó-böngészõ dokumentációja

A GNOME súgó-böngészõ egy felület, amelyen a számítógépeden megtalálható Külömbözõ formátumú dokimentációk, valamint az interneten fellelhetõ anyagok egyaránt megtekinthetõk. Ez a kis leírás a legáltalánosabb tudnivalókra hívja fel a figyelmet.

Parancssori opciók

      -d, --debug                Nyomkövetési szint, nagyobb szám=több üzenet
  -h HEIGHT                  Az ablak szélessége
  -w WIDTH                   Az ablak magassága
  -x X                       Az ablak X pozíciója
  -y Y                       Az ablak Y pozíciója
      --display=DISPLAY      A használandó X képernyõ
      --no-xshm              Ne használja az 'X shared memory extension'-t
  -?, --help                 Ez a lista
      --usage                Rövid használati utasítás
  -V, --version              A program verziószáma

Menük

Van egy pár menü a GNOME súgó-böngészõ felsõ részén, ezek viszont annyira egyértelmûek, hogy magyarázni fölösleges.

Eszközsor

A gombsor amely a menük alatt található, könnyebben elérhetõ utasításokat nyújt neked. Példa kedvéért az Elõzõ vagy a Következõ gomb az elõzõ vagy a következõ oldalra ugrik, ha már nézegetted õket ennél a futtatásnál. A Súgó egy súgó a súgóról, de bizonyára ezt már tudod, hisz ezt a dokumentumot hozza be. A Napló gomb az eddig látogatott lapok listáját hozza be külön ablakban, rákattintva egy-egy hivatkozásra a fõablakban megjelenik a hivatkozott oldal. A Könyvjelzõk gombbal elõ tudod szedni azoknak az oldalaknak a listáját, amelyeket fontosnak gondoltál.

URL beviteli mezõ

Ez egy sima beviteli mezõ, ahova URL-eket, vagyis weblap-helyeket, vagy speciális dokumentum-megnevezéseket írhatsz. A következõ URL típusokat érti:

http:
A HTTP protokollnak megfelelõ megnevezéseket.

Példák:
  • http://www.gnome.org
  • http://www.gnome.org/index.shtml
  • http://www.gnome.org/index.shtml#majic
file:
A fájlrendszerben megtalálható dokumentumokat.

Példák:
  • file:/home/msf/maji12.html
  • file:/usr/man/man1/ls.1
  • file:/usr/src/gogo/README
whatis:
Egy dokumentum nevének részletére keresést.

Példák:
  • whatis:ls
  • whatis:xman
toc:
Különbözõ dokumentumok tartalomjegyzékét.

Példák:
  • toc:
  • toc:man
  • toc:info
  • toc:ghelp
man:
Kézikönyv lapokat (unix manual, man).

Példák:
  • man:ls
  • man:ls(1)
info:
GNU info lapokat.

Példák:
  • info:emacs
  • info:emacs#Top
ghelp:
GNOME súgót egy-egy alkalmazáshoz.
Megjegyzés:Nem sok minden van eddig kész ebben a formátumban...

Példák:
  • ghelp:help-browser

Könyvjelzõ ablak

A könyvjelzõ ablak mint már említettem egy lista azokról az oldalakról, amiket érdekesnek találtál és a Könyvjelzõ hozzáadása paranccsal megjegyeztél. Rákattintva egy-egy bejegyzésre a fõablakban meg fog jelenni az oldal.

Napló ablak

A napló ablakról szintén volt már szó, lényege, hogy az összes látogatott lap bekerül egy listába, ebbõl tudsz válogatni. Van egy maximális hossza, amely a Beállításokban állítható át.
