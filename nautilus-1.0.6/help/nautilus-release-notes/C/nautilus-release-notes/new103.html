What's New in Nautilus 1.0.3

Nautilus 1.0.3 has many improvements over previous Nautilus releases. Many of these are new features and other changes which were suggested by Nautilus users. Nautilus 1.0.3 also has significant performance improvements and many bug fixes. Some of the major improvements are described below.

Major new features and improvements:

  • New News sidebar downloads headlines off of your favorite news sources.

  • Improved and faster audio preview.

  • Improved integration with GNOME desktop and GNOME Panel.

  • Improved internationalization.

  • New Script feature allows users to run any script on selected files.

  • New Cut, Copy, and Paste feature for moving and copying files.

  • New Delete feature which bypasses trash.

  • Support for Emacs bindings, which can be enabled in Preferences dialog.

  • Identifies if files were changed outside of Nautilus and updates views accordingly.

  • Notes sidebar identifies whether a note exists for the directory by displaying an icon in the tab.

  • Tree view is faster and does not use processor when closed.

  • Improved font selection dialog.

  • New global view preferences.

  • Support for HTTP proxies which require username and password.

  • Improved help sidebar appearance.

  • Improved compatibility with gmc.

  • And many other improvements to make Nautilus more intuitive and fun to use.
