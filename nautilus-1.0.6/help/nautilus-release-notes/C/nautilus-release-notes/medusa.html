Building Nautilus and Medusa

Medusa is an indexing program which produces an index of all the files and their contents on the computer. Because there were a couple significant problems caused by Medusa, it is recommended that Medusa be disabled.

Medusa support is automatically built into Nautilus at configure time if Medusa is on the machine. Thus, if you are repackaging Nautilus, you should be sure that your machine does not have Medusa installed on it.
