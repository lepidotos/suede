System Requirements

Red Hat Linux 6.1 or 6.2: An IA-32-compatible system with a minimum of 32MB of RAM.

Red Hat Linux 7: The latest GNOME 1.2 or 1.4 libraries running on an IA-32-compatible system with a minimum of 32MB of RAM.
