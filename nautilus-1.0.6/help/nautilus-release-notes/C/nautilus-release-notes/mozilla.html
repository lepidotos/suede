Mozilla

It is recommended that Nautilus 1.0.4 be used with Mozilla 0.9.1. If this is not possible, Nautilus will probably work with 0.8. Mozilla 0.8.1 has a known problem which causes the links in help documents in Nautilus to not work, so this version should be avoided.
