What's New in Nautilus 1.0.4

Nautilus 1.0.4 has many small bug fixes and performance improvements. There are also a few new features which have been added since version 1.0.3. Some of the more notable improvements are listed below.

Major new features and improvements:

  • Improved internationalization.

  • Updated icons.

  • Improved keybindings using Bonobo.

  • Better support for XFS file system.

  • New preference to have all thumbnails stored under home folder.

  • And many other small bug fixes and performance improvements.
