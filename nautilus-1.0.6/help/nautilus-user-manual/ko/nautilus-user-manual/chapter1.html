노틸러스 소개

노틸러스는 그놈 바탕화면 환경의 핵심 구성요소 입니다. 노틸러스는 파일과 폴더, 웹 찾아보기 등을 보고, 관리하고 사용자의 입맛대로 바꿀수 있는 쉬운 방법을 제공합니다.

노틸러스는 파일, 응용프로그램, 미디어, 인터넷 기반 자원, 웹으로의 접근을 통합시켜 주고, 사용할수 있는 모든 자원을 쉽고 빠르게 가져와서 사용할수 있게 하여줍니다.

메뉴 막대

대부분의 파일과 폴더 관리 작업을 할때에는 8개의 메뉴 옵션이 있습니다. 그리고 모든 하드 디스크의 응용프로그램, 폴더, 파일 등을 마음대로 고칠도록 할수 있습니다. 도움말 메뉴 오른쪽의 세부설정 메뉴는 기호로 표시되며 리눅스와 그놈의 사용경험의 수준을 선택할수 있도록 하여줍니다.

도구 막대

도구막대는 한번 누르기로 컴퓨터의 디렉토리와 웹을 찾아보고 검색할수 있게 하여줍니다. 서비스 버튼을 누르면 리눅스 사용자에게 필요한 서비스의 모음인 Eazel서비스로 바로 접근할수 있도록 하여줍니다.

위치 막대

위치 막대는 경로 이름, 웹 주소(URL) 또는 다른형식의 주소들을 입력 할기위해 사용합니다. 오른쪽 끝의 팝업 메뉴는 다양한 보기를 - 예를 들어 아이콘으로 보기나 목록으로 보기 등등. - 선택할수 있도록 하여 줍니다. 더하기와 빼기 표시를 눌러서 아이콘의 크기를 늘이거나 줄일수 있고 돋보기를 누르면 원래 크기로 돌아옵니다(테마를 바꾸었다면 돋보기 기호는 다른 기호로 바뀌어 질수도 있습니다.)

가장자리 막대

가장자리 막대는 현재파일이나 폴더에 대한 정보를 표시합니다. 가장자리 막대의 바닥에 있는 4개의 탭은 각각 부가적인 정보나 도움말을 제공합니다:

  • 나무구조 탭은 나무구조를 표시합니다 - 컴퓨터의 폴더와 파일 구조를 계층적으로 보여줍니다

  • 기록 탭은 이전에 방문하였던 경로나 주소의 위치를 목록화 하여 줍니다

  • 도움말 탭은 설명서, 그놈 정보 페이지, 리눅스 설명서 페이지 등등 의 모든 정보 자원에 빨리 접근할수 있도록 하여 줍니다

  • 노트 탭은 현재 폴더에 대한 간단한 메모나 해야할일, 기록 상태 등등 을 남길수 있는 공간을 제공합니다

주의: 사용자 수준이 초급이거나 가장자리 막대 세부사항을 바꾸지 않았다면 가장자리 막대의 4가지 탭을 모두 못볼수 있습니다.

상태 막대

상태 막대는 메뉴 항목에 대한 정보를 표시합니다. 메뉴 항목에 대한 설명을 보려면, 메뉴에 마우스 포인터를 두고 상태바의 글씨를 읽으 십시오.

주 패널

노틸러스 창의 주 패널은 탐색의 대부분을 여기서 할수 있습니다. 파일, 폴더, 응용프로그램들은 여기서 표시 됩니다. 보기를 변경하거나 바꿀수 있는 몇가지 옵션이 있습니다.
