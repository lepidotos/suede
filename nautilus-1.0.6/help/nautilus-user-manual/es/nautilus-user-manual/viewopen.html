Visualizando y abriendo archivos

Vistas Icono y Lista

La primera vez que ejecuta Nautilus puede ver las carpetas y los archivos representados como iconos. Esta es la vista icono.

Intente lo siguiente:
 

Observando sus archivos y carpetas de dos formas:

  • Para ver los contenidos de una carpeta como lista haga clic en el menú emergente Ver como y elija Ver como Lista.

  • Para ordenar los elementos mostrados en la vista lista, haga clic en las cabeceras de las columnas (Nombre, Tamaño, Tipo y Fecha de Modificación).

  • Para volver a la vista icono, abra el menú emergente Ver como y elija Ver como Iconos.

  • Para ordenar los elementos en la vista icono, abra el menú Ver. Luego, elija Ordenar Elementos y después elija una opción de ordenamiento. (Para aprender más acerca de las opciones de ordenamiento vea Seleccionando el Orden de Elementos.)
