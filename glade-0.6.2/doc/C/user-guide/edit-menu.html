Edit menu

Figure 3. Edit menu

Cut

This command will remove a widget from its parent and place it on the clipboard.

Copy

This command will copy a widget to the clipboard. The original widget remains attached to its parent.

Paste

If a widget currently exists on the clipboard, this command will allow you to copy the widget from the clipboard

Clear

If a widget is currently selected, this command will delete it from its parent without moving it to the clipboard.
