Grid Options

Figure 6. Grid options

  • Horizontal spacing - number of pixels to skip horizontally before drawing a line

  • Vertical spacing - number of pixels to skip vertically before drawing a line

  • Grid style - should the grid be drawn as a series of dots or as lines?

Snap Options

Figure 7. Snap Options

  • Horizontal snapping - should Glade snap the left and/or right edges of a widget to the grid?

  • Vertical snapping - should Glade snap the top and/or bottom edges of a widget to the grid?
