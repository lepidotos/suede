Invia la notifica

Una volta completata la notifica, la si può rivedere e decidere cosa farne. Probabilmente si vorrà inviarla al sistema di bug tracking, ma se per qualche motivo la si vuole inviare da qualche altra parte, lo si può fare. Si può scegliere fra:

Invia la notifica

Selezionando questa possibilità e cliccando su Succ si invierà la notifica al bug tracker. Si può inviarne una copia a se stessi attivando l'apposita checkbox. Se si invia la notifica ad un bug tracker indipendente, sarà necessario inserire a mano l'indirizzo di destinazione.

Invia la notifica solamente al proprio indirizzo di posta

Questa invia la notifica a se stessi. Se si è modificato l'indirizzo email all'inizio, allora sarà usato il nuovo indirizzo.

Salva la notifica in un file

Selezionando questa possibilità verrà visualizzata una casella per inserire il nome di un file nel quale salvare la notifica dell'errore. Se il file esiste già, verrà chiesto se si vuole sovrascriverlo.
