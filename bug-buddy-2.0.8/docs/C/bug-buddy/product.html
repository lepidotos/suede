Product

This page gives you a list of products, from which you can select, and a button to pick the bug-tracker to use. If you select a new bug-tracker, any highlighted product will become unhighlighted again.

The choices currently are to select All systems, or to select Gnome's system. The most sensible one to choose is Gnome's.
