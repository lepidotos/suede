Contact Information

Bug Buddy first asks for these pieces of information, in order to be able to send the report correctly. It makes very good guesses so you should be able to skip straight ahead.

Your name

Bug Buddy works this out itself, but if you want to change it you can do this here.

Email address

Please enter a valid email address for you here. This is so developers can request more information from you, or inform you of a fix for your bug.

Full path of sendmail

This allows you to check that Bug Buddy is looking in the right place when it tries to send email. You will almost certainly not need to change this.
