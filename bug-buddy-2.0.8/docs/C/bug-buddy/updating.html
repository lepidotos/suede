Updating Product Listing

If this is the first time you have run Bug Buddy, you will see will see this page. It also appears if it has been two weeks since the last update. Otherwise it will not appear.

A dialogue box telling you Bug Buddy wants to update its list of information will appear and gives you the option to proceed or to say no. It will save this information by default into your .gnome/Bug Buddy.d/ directory. To get this information, it needs to talk to bugzilla, so your net connection will have to be up.
