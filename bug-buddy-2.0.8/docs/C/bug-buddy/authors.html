Bug Buddy was written by Jacob Berkman <jacob@bug-buddy.org>. To find more information about it, please visit Bug Buddy's home on the web.

Some parts of this document (the definitions of bug severity) are directly lifted from the new Gnome bugtracker mentioned above.
