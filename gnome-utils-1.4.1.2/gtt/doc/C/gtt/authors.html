Authors

GNOME Time Tracker was written by Eckehard Berns and Linas Vepstas. It is currently maintained by Linas Vepstas <linas@linas.org>. Please send all comments, suggestions, and bug reports to Linas, as well as to the GNOME bug tracking database. Instructions for submitting bug reports can be found on-line. You can also use the GNOME Bug Report Tool (bug-buddy), available in the Utilities submenu of Main Menu, for submitting bug reports.

There is a webpage devoted to the latest GTT news at http://www.linas.org/linux/gtt/gtt.html.

This manual was written by Aaron Weber (<aaron@helixcode.com>) Eckehard Berns (<eb@berns.i-s-o.net>) and Linas Vepstas (<linas@linas.org>) Please send all comments and suggestions regarding the manual to the GNOME Documentation Project at <docs@gnome.org>. You can also add your comments online by using GNOME Documentation Status Table.
