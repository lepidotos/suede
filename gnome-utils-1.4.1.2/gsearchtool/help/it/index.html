Introduzione

Il Cerca File di GNOME è un'utilità per cercare i file nel proprio sistema. Può effettuare molti tipi di ricerca: per nome di file, per data di modifica, per contenuto del file, ecc. È incluso nel pacchetto gnome-utils, che fa parte dell'ambiente desktop GNOME. Questo documento descrive la versione 1.2 dell'applicazione Cerca File di GNOME.

Il programma può essere avviato selezionando Cerca file nel sottomenu Utilità del Menu principale, oppure lanciando il comando gsearchtool da riga di comando.

Si noti che il Cerca File di GNOME è semplicemente un'interfaccia ai comandi UNIX find, grep e locate. Questi comandi sono estremamente potenti, ma non molto facili da usare, specie per un nuovo utente. Il Cerca File di GNOME fornisce un'interfaccia semplice per gli usi più comuni di questi comandi. Se si debbono effettuare alcuni tipi di ricerca che non sono gestiti dal Cerca File di GNOME, è probabile che li si debba fare da riga di comando utilizzando find e grep con le opzioni appropriate. In questo caso si dovrebbero leggere le pagine info di questi comandi, ad esempio utilizzando la Guida di GNOME.
