Manuale del Cerca File di GNOME
IndietroAvanti

Uso

Il Cerca File di GNOME mette a disposizione due tipi di ricerca: la Ricerca completa e la Ricerca veloce. La ricerca completa è affidabile e fornisce molte possibilità, ma può essere lenta. La ricerca veloce usa una tecnica speciale per velocizzare la ricerca; è molto più veloce ma può cercare i file solo in base al nome e non è così accurata come la ricerca completa. Questa sezione descrive la modalità di ricerca completa; la ricerca veloce è descritta nel.

Figura 1. Finestra principale di ricerca

Per avviare una ricerca, si deve dare al Cerca File di GNOME il criterio di ricerca e la directory da cui partire.

Per default, la ricerca parte nella directory di lavoro corrente; se il Cerca File di GNOME è stato avviato da menu, allora la directory di lavoro corrente sarà la propria home directory. Si può modificare la directory di partenza inserendo un percorso valido nel campo Comincia nella directory, oppure scegliendo una directory usando il pulsante Sfoglia.... Per esempio, per cercare attraverso tutti i file nel proprio sistema si inserisca / in questo campo (attenzione: questa operazione durerà molto tempo).

Per default, il Cerca File di GNOME cerca inoltre attraverso tutte le sottodirectory della directory specificata; si può disabilitare questo comportamento attivando l'opzione Non cercare nelle sottodirectory. Per un controllo più fine si può abilitare l'opzione Non cercare nei filesystem montati, che escluderà dalla ricerca le sottodirectory che sono fisicamente presenti (montate) su dischi o partizioni diverse da quella della directory di partenza si veda per i dettagli.

Il solo criterio di ricerca mostrato per default è per Nome file. In questo campo si possono usare i caratteri jolly * (corrisponde a qualsiasi sequenza di simboli, compreso quella vuota) e ? (corrisponde a un qualsiasi simbolo). Per esempio, per trovare i file con estensione txt, si inserisca *.txt in questo campo; per trovare tutti i file che cominciano con la lettera D, si inserisca D* (attenzione: i nomi dei file sono sensibili alle maiuscole/minuscole!). Si possono anche specificare criteri di ricerca più complicati si veda .

Dopo aver specificato il criterio di ricerca e la directory di partenza, si prema Cerca. Ci metterà un po' di tempo (quanto tempo dipende dalla dimensione della directory e dalla velocità del sistema), dopo il quale sarà mostrata una finestra contenente la lista di tutti i file che corrispondono al proprio criterio (potrebbe essere vuota), tipo quella mostrata in . Se la ricerca ci mette troppo tempo, la si può bloccare premendo il pulsante Interrompi.

Figura 2. Finestra dei risultati della ricerca

Nella finestra dei risultati della ricerca, cliccando due volte su un qualsiasi file lo si può aprire. Si otterrà lo stesso effetto che cliccando due volte sullo stesso file in un File Manager cioè, il file è aperto usando l'azione predefinita per questo tipo di file. Premendo sul pulsante Salva si può anche salvare su disco la lista dei file trovati. Verrà chiesto di specificare il nome del file nel quale salvare la lista; la lista sarà salvata in normale formato testo.
