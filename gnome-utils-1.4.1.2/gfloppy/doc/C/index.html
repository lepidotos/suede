Introduction

GFloppy is a simple floppy formatter for Linux. You may need to format your floppies when they are new.

To run GFloppy, select gfloppy from the Utility submenu of the Main Menu, right click on the floppy icon on your desktop and select Format Floppy, or type gfloppy on the command line.

This document describes version 0.9.4 of GFloppy.
