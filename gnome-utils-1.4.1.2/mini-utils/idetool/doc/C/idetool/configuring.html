Configuring IDE Devices

Some IDE properties(DMA Mode, IRQ Unmask, and On Reset) can be configured using IDE Tool. To modify the setting of any of these properties, just double-clicking on the item. This will toggle the setting for the particular property.

Warning: Only expert users should change the IDE settings
 

Mis-configuration of IDE devices can cause serious malfunctions of the system. Make sure to learn the proper use of each setting and the common pitfalls in configuration. The man pages for the hdparm command can be very instructive for learning more about this. You can view these man pages by clicking on the link above or by typing man hdparm at a shell prompt.
