Introduction

Guname displays a variety of basic information about your system.

To run Guname, select System info... from the System submenu of the Main Menu, or type guname on the command line.

Guname is included in the GNOME-utils package, which is part of the GNOME desktop environment. This document describes version 1.2 of Guname.
