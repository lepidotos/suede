Introduction

Gw displays a listing of all the users currently logged onto your machine, along with information about what they're up to. It also gives you the ability to communicate with them.

To run Gw, select User Listing from the System submenu of the Main Menu, or type gw on the command line.

Gw is included in the GNOME-utils package, which is part of the GNOME desktop environment. This document describes version 1.4 of Gw.
