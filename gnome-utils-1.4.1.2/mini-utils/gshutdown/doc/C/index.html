Introduction

Gshutdown is a program for shutting down your computer. It is included in the package GNOME-utils, which is a part of the GNOME desktop environment. This document describes version 1.2 of Gshutdown.

Gshutdown can be started by selecting Shut Down or Reboot from System submenu of the Main Menu, or by running the command gshutdown on the command line. Gshutdown is intended for use by system administrators, and must be run with root authority.
