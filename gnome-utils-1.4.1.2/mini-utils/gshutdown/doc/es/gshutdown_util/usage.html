Utilizando Gshutdown

Utilización Básica

Cuando se inicia por primera vez Gshutdown, podrá ver la Ventana principal, tal como .

Figura 1. Ventana Principal de Gshutdown

Esta es la ventana de apagado de Gnome. Si desea apagar su ordenador, seleccione "Apagar", en vez de "Reiniciar," entonces presione "OK." Espere unos pequeños instantes para que su ordenador se prepare, entonces apágelo. Si Vd. observa la línea "System Halted," podrá apagar el sistema de forma segura. Si su ordenador admite la Gestión Avanzada de Energía (APM), como muchos ordenadores portátiles y algunos de escritorio, no necesitará el botón de apagado.

Para reiniciar, simplemente elija "Reiniciar," presione "OK," y espere.

Capacidades Avanzadas

Figura 2. Ventana Avanzada de Gshutdown

Vd puede cambiar a cualquier nivel de ejecución del sistema en la pestaña Avanzada.

Un nivel de ejecución (Runlevels) es una configuración de software del sistema que le permite seleccionar el grupo de procesos que van a existir. En el fichero /etc/inittab se define el crecimiento de los procesos para cada nivel de ejecución causado por init. Init puede tener 8 niveles distintos de ejecución: 0-6 y S ó s.

Se reservan los niveles de ejecución 0, 1, y 6. El nivel 0 se utiliza para parar el sistema, el nivel 6 para reiniciar el sistema, y el nivel 1 para llevar al sistema al estado mono usuario. El nivel S no significa que se utilice directamente, la mayor parte de los guiones ("scripts") lo invocan cuando entran al nivel 1.

En la init página del manual hay más información disponible sobre los niveles de ejecución.

Preferencias

Figura 3. Ventana de Preferencias de Gshutdown

Existe también un diálogo de preferencias que le permite configurar el apagado de la línea de comandos. Para los detalles de los niveles de ejecución y el efecto que las instrucciones de apagado de la línea de comandos observe la documentación de su ordenador. Si no está familiarizado con estos detalles, puede presionar la pestaña Fácil. Si tiene conocimientos sobre estas cuestiones, la pestaña Avanzada le será auto-explicativa.
