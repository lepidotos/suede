Introduction

GNOME Archive Generator makes compressed tape archive record (tar) files. A tar file is a single bundled file that contains a directory or group of files.

To run GNOME Archive Generator, select ->Utilities+ + ->Archive Generator+ + + + + + + from the Main Menu, or type meat-grinder on the command line.

GNOME Archive Generator is included in the gnome-utils package, which is part of the GNOME desktop environment. This document describes version 1.4.0.1 of GNOME Archive Generator.
