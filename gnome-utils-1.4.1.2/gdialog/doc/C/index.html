Introduction

gdialog is a command prompt application which displays GNOME dialog boxes. It is meant to be used as a quick way to display dialog boxes within shell scripts.

gdialog helps you to create small and useful commonly used dialog boxes, like message boxes, information boxes, input boxes, etc.

gdialog is included in the gnome-utils package, which is part of the GNOME desktop environment. This document describes version 0.6gnome1 of gdialog.
