Introduction

GDiskFree is a program for monitoring disk usage on your computer. It is included in the GNOME-Utils package, which is a part of the GNOME desktop environment. This document describes version 1.4 of GDiskFree.

You can start GDiskFree by selecting GNOME DiskFree from the Utilities submenu of the System Menu, or by running the command gdiskfree on the command line.
