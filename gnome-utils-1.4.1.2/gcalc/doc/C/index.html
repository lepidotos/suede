Introduction

GNOME Calculator is a double precision calculator application. It provides advanced scientific and mathematical features such as trigonometric functions (sine, cosine, tangent) and logarithms (log, ln).

To run GNOME Calculator, select Simple Calculator from the Utilities submenu of the Main Menu, or type gcalc on the command line.

GNOME Calculator is included in the gnome-utils package, which is part of the GNOME desktop environment. This document describes version 1.2.0 of GNOME Calculator.
