Authors

GDict was written by Spiros Papadimitriou (<spapadim+@cs.cmu.edu>, Mike Hughes (<mfh@psilord.com>) and Bradford Hovinen (<hovinen@helixcode.com>). To find more information about GDict, please visit the GDict Web page. Please send all comments, suggestions, and bug reports to the GNOME bug tracking database. You can also use Bug Report Tool (bug-buddy), available in the Utilities submenu of Main Menu, for submitting bug reports.

This manual was written by John Fleck(<jfleck@inkstain.net>). Please send all comments and suggestions regarding this manual to the GNOME Documentation Project by sending an email to <docs@gnome.org>. You can also add your comments online by using the GNOME Documentation Status Table.
