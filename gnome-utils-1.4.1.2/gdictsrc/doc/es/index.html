Introducción

GDict proporciona definiciones de diccionario para las palabras, para ello emplea cualquier servidor que permita el protocolo dict, un estándar de Internet para aplicaciones de diccionario cliente/servidor.

Para ejecutar GDict, seleccione GDict desde el submenú Utilidades del Menú Principal, o teclee desde la línea de comandos gdict -a.

También es posible arrancar GDict como un aplique del panel. Para añadir este aplique al Panel, pulse con el botón derecho sobre el panel y elija +Apliques+ + ->Utilidades+ + ->GDict+ + + + + + + , o teclee desde la línea de comandos gdict.

GDict se incluye en el paquete gnome-utils, el cual forma parte del entorno de escritorio GNOME. Este documento describe la versión 1.2 de GDict.
