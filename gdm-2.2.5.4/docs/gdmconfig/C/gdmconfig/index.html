Introduction

GDM Configurator, (GNOME Display Manager Configurator) is a configuration application for the GDM (Gnome Display Manager). Typically, when running your system in a graphical login mode, the GDM is the application that facilites logins. GDM Configurator allows for easy configuration of the GDM at various levels of detail. Customizations affect both operations and appearances.

To run GDM Configurator, select System+ + + + + ->GDM Configurator+ + + + + from the Main Menu, or type gdmconfig on the command line.

Root access required
 

GDM Configurator requires root access to run.

GDM Configurator is included in the Gnome-Core package, which is part of the GNOME desktop environment. This document describes version 2.0.97.1 of GDM Configurator.
