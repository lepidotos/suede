Autori

Il Mixer di GNOME è stato scritto da Jens Ch. Restemeier <jchrr@hrz.uni-bielefeld.de>. Per ulteriori informazioni sul Mixer di GNOME, visita la pagina web di Gmix. Invia ogni commento, suggerimento e segnalazione di bug allo GNOME bug tracking database. (Puoi trovare on-line le istruzioni su come presentare segnalazioni di bug). Se usi GNOME 1.1 o successivo, per inviare segnalazioni di bug puoi anche usare il Bug Report Tool (bug-buddy), disponibile nel sottomenu Utilità del Menù Principale.

Questo manuale è stato scritto da Erik Jarvi <ejarvi@megsinet.net>. Invia ogni commento e suggerimento riguardante questo manuale allo GNOME Documentation Project spedendo una e-mail a <docs@gnome.org>. Puoi anche aggiungere on-line i tuoi commenti usando la GNOME Documentation Status Table.

Traduzione italiana a cura di Salvo Isaja (<salvois@it.gnome.org>), a cui può essere inviato ogni commento e suggerimento riguardante questa traduzione.
