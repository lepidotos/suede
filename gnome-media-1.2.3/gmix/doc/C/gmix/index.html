Introduction

The application GNOME Mixer (gmix) is a GNOME enabled audio mixer. Gmix allows you to mix audio for 2 channel sound cards.

To run GNOME Mixer select + ->Multimedia+ + + ->Audio Mixer+ + + from the Main Menu, and can also be invoked in the command line with $ gmix.
