Introduction

The GNOME CD Player (gtcd) is a GNOME enabled application that is preloaded with GNOME. This is a simple CD Player which allows you to listen to compact discs on your PC.

To run GNOME CD Player select + ->Multimedia+ + + ->CD Player+ + + from the Main Menu, and can also be invoked in the command line with $ gtcd.
