Chapter 3. Menu options

  • File menu

    • New Window. Create a new window in which to display a PostScript document

    • Open. Open a new file.

    • Reload. Read the file again and update the screen accordingly.

    • Print. Print document.

    • Print Marked Pages. Print only those pages which are marked.

    • Save Marked Pages. Save marked pages to another files.

    • Close. Close current window. If it is the only one, then quit the program.

    • Exit. Quit GGv.
