Capitolo 4. Impostare le preferenze di GGv

La finestra di dialog delle preferenze è suddivica in quattro regioni: Documento, Layout, Ghostscript e Stampa.

Nella sezione Documento è possibile impostare il formato del documento se non è specificato esplicitamente in quest'ultimo. Se viene selezionata l'opzione Ignora il formato della pagina, verrà utilizzato il formato che avete scelto, ignorando quello, se presente, fornito dal documento stesso. Il tipo di formato viene espresso tramite i comuni formati della carta (come A4, Letter e così via). L'opzione Antialias permette di sfrutta il supporto di Antialiasing di Ghostscript durante la visualizzazione. L'antialias migliora notevolmente la qualità della visualizzazione del documento.

La sezione Layout vi permette di scegliere se visualizzare o meno la barra del menu oppure il pannello laterale di GGv. Questi vengono mostrati se si utilizza l'impostazione predefinita.

È possibile specificare il comando che GGv utilizza per chiamare l'interprete Ghostscript nella sezione Ghostscript. L'interprete deve contenere il percorso completo dell'eseguibile di Ghostscript con, in aggiunta, tutte le strane e assurde opzione che desiderate utilizzare. Leggi PDF deve contenere il percorso completo per Ghostscript più le opzioni per leggere i file in formato PDF e infine Decomprimi deve contenere il percorso completo all'eseguibile utilizzato per decomprire i file postscript compressi con gzip. Lasciate queste opzioni invariate se non sapete esattamente cosa state eseguendo.

Nella sezione Stampa è possibile specificare il comando utilizzato per stampare il documento. Il nome del file viene passato al comando utilizzando la stringa %s.
