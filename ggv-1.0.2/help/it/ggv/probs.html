Capitolo 5. Possibili problemi con GGv

Se non vi è possibile caricare un file postscript, controllate l'inizio del file (decomprimete, se necessario). Il file può non essere in formato postscript oppure è stato creato su di un sistema che utilizza caratteri differenti per la terminazione della riga diverso da quello di Unix (Ad esempio, MacOS). La soluzione più semplice per quest'ultimo problema è quello di creare un piccolo script in perl che converta il file nel formato corretto.
