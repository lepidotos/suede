Capitolo 2. GGv: la finestra principale

Ogni documento risiede in una finestra principale. Questa consiste in una barra di menu, pulsanti per lo zoom e la navigazione tra le pagine, la lista delle pagine disponibili e infine la zona di visualizzazione del documento postscript.

  • Vista principale. Questa area delle finistra è dove viene visualizzato il documento. Se il vostro schermo è sufficientemente grande oppure il documento è piccolo quanto basta, in questa area verrà mostrata la pagina completa del documento, altrimenti una sola sezione di questa.

  • Lista della pagine. La lista dei numeri in basso a sinistra mostre tutte le pagine disponibili nel documento postscript visualizzato.

È possibile spostarsi all'interno del documento in svariati modi: i tasti cursore possono essere utilizzati per spostarsi nella direzione desiderata, la barra spaziatrice sposta il documento nel senso di lettura (assume che il verso sia da sinistra a destra e dall'alto in basso) e il tasto backspace sposta nella direzione inversa. È inoltre possibile spostarsi trascinando il documento con il mouse, cliccando e spostando la vista.

Cliccando sulla vista principale, con il tasto destro del mouse, verrà mostrato un menu a tendina, il quale permette di navigare tra le pagine, ingrandire o ridurre l'immagine, ricentrare la pagina oppure scegliere se mostrare o meno la barra del menu o il pannello laterale.

È inoltre possibile trascinare una lista di file dal file manager dentro la finestra principale di GGv, il quale aprirà e visualizzerà in finestre differenti, una per file.

Dal menu File è possibile aprire una nuova finestra di visualizzazione, aprire un nuovo documento oppure ricaricare un già aperto. È inoltre possibile chiudere la finestra (GGv terminerà se non rimangono finestre da chiudere) oppure terminare l'applicazione.

Il menu Documento vi permette di spostarvi tra le pagine, avanti o indietro, ricentrare la page oppure impostare l'orientamento o lo zoom delle pagine visualizzate.

Nel menu Impostazioni è possibile aprire una finestra di dialogo per le Preferenze oppure scegliere se mostrare o meno la barra del menu oppure il pannello laterale.
