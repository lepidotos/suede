Capitolo 3. Opzioni del Menu

  • menu File

    • Nuova finestra. Crea una nuova finestra dove aprire un documento PostScript.

    • Apri. Apre un nuovo file.

    • Ricarica. Legge nuovamente il file dal disco e aggiorna lo schermo se necessario.

    • Stampa. Stampa il documento.

    • Stampa le pagine marcate. Stampa unicamente quelle pagine che sono state precedentemente marcate.

    • Salva le pagine marcate. Salve le pagine precedentemente marcate in un nuovo file..

    • Chiudi. Chiude le finestra corrente. Se una sola finestra è presente, l'applicazione terminerà.

    • Esci. Esce da GGv.
